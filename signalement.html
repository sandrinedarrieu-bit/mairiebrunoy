Comment ça fonctionne en conditions réelles

Ce que fait l'habitant : remplit ce formulaire depuis son téléphone ou ordinateur. Reçoit immédiatement une confirmation d'inscription ou de mise en liste d'attente selon les places disponibles.

Ce que font les agents : les inscriptions arrivent automatiquement dans leur base de suivi, sans ressaisie. Les listes d'attente se gèrent automatiquement par ordre d'arrivée.

Pourquoi cet outil : le signalement d'incident existe déjà sur le site de Brunoy. Ce formulaire couvre un besoin non encore digitalisé — les inscriptions aux activités du service Brunoy Familles.
